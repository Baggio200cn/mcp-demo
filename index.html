理解 MCP 架构

Model Context Protocol (MCP) 是连接 AI 应用与外部工具的桥梁
让我们通过一个生动的比喻来理解它的工作原理

顾客

你的应用 / Claude

  • 想要点餐（需要功能）
  • 不会做菜（不能直接操作）
  • 只会看菜单（发送请求）
📱

服务员

MCP 客户端

  • 懂顾客语言（理解需求）
  • 与厨房沟通（协议转换）
  • 传递订单（数据传输）
👨‍🍳

厨房

MCP 服务器

  • 有厨具食材（工具数据）
  • 会做菜（实现功能）
  • 规范订单（标准协议）
